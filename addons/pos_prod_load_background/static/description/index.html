Odoo 10 Point of Sale Extension

Loading products and customers in background to improve speed of Point of Sale front-end.

  • When user starts pos session at that time all products and customers are load into background.
  • We can access new products which is newly created into back-end without load point of sale front-end.
  • We can search partners while partners are loading.
  • After load partners we can also access new partners without load point of sale front-end.
